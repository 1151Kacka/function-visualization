co program dělá
    využívá knihovny numpy a matplotlib k vizualizaci grafů definovaných funkcí

jak ho spustit
    nainstalováním si potřebných knihoven a kliknutím na tlačítko run ve vs, nebo zadáním python main.py do terminálu vs
jaké funkce vykresluje
    lineární, kvadratická, sinusová, exponenciální a logaritická 
a hledá jejich minima a maxima
